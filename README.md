QCost_0.8.1_win32_bin.zip contains binaries for windows platform. Uncompress the file and click on QCost.exe to launch the program. These binaries were created with msvc 32-bit. Probabily this is the most common solution on windows platform. QCost_0.8.1_win32_64_bin.zip contains binaries compiled with msvc 64-bit. QCost_0.8.1_win32_mingw32_bin.zip contains binaries compiled with mingw 32-bit.

QCost_0.8.1_src.tar.gz containes source code. To build binaries you need library Qt>=5.3. 
Uncompress the file and launch
  $ qmake
  $ make
You should have the binaries in folder bin.

If you encounter problems post a message on http://ingegnerialibera.altervista.org/forum/viewforum.php?f=3

Reference site is http://ingegnerialibera.altervista.org/wiki/doku.php/qcost:indice.

QCost_0.8.1_win32_bin.zip contiene gli eseguibili per Windows. Decomprimi il file e clicca su QCost.exe per lanciare il programma. Questi binari sono stati creati con msvc versione a 32-bit. Molto probabilmente questa è la soluzione più comune per windows. QCost_0.8.1_win32_64_bin.zip invece contiene eseguibili copmilati con msvc 64-bit. Infine QCost_0.8.1_win32_mingw32_bin.zip contiene l'eseguibile di QCost compilato con mingw 32-bit.

QCost_0.8.1_src.tar.gz contiene il codice sorgente. Per creare l'eseguibile devi avere installato le librerie Qt, almeno la versione 5.3. 
Scompatta il file e lancia i comandi
  $ qmake
  $ make
Se la compilazione va a buon fine dovresti trovare gli eseguibili nella cartella bin.

In caso di problemi lascia un messaggio sul forum di QCost all'indirizzo http://ingegnerialibera.altervista.org/forum/viewforum.php?f=3.

Per maggiori informazioni il sito di riferimento è http://ingegnerialibera.altervista.org/wiki/doku.php/qcost:indice.
